# **App Name**: ShopWise Landing

## Core Features:

- Deal Display: Display product image with a 'Best Deal Found!' banner.
- Price Comparison: List price comparisons with the lowest price highlighted.
- Realistic Rendering: Utilize photorealistic rendering to showcase the app on a modern smartphone.

## Style Guidelines:

- Primary color: Vibrant green (#32CD32) to highlight the 'Best Deal'.
- Background color: Light, desaturated green (#E0F8E0) for a clean look.
- Accent color: Orange (#FFA500) for calls to action and emphasizing discounts.
- Font: 'Inter' sans-serif for clear and modern UI text.
- Note: currently only Google Fonts are supported.
- Minimalist desktop setting with soft focus background.
- Use clean, simple icons for clarity in the price comparison list.
